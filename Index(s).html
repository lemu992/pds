Sobre o FL

Bem-vindo ao FL - Studio!

O FL Studio é uma ferramenta que permite criar e editar músicas disponível para download no PC (Windows ou macOS), além de ser possível baixar a versão mobile para celulares Android ou iPhone (iOS)..

O FL Studio é um mundo! Realmente é um software muito robusto e cheio de plugins gratuitos que permitem elevar as produções de áudio, sendo muitos desses plugins produzidos por grandes referências do mundo fonográfico. Já utilizei a versão free e algumas vezes a versão paga do estúdio de produção do meu irmão e tem várias funcionalides. Vire e mexe tem umas promos, o que permite que o software tenha um preço mais acessível.

Requisitos do Sistema

- P2Ghz Intel Pentium 4/AMD Athlon 64 (ou superior) -

- 4GB ou mais de RAM -

- 4GB livre no Disco Rigido -

- Placa de Som com drivers DirectSound e compatibilidade com ASIO/ASIO2 para gravação de áudio -

- Windows XP (SP3), Windows Vista, 7/8/8.1/10/11 - 32 ou 64 bits -

- iMac com Boot Camp para Windows (mesmas versões acima) -

- ou iMac com OS X 10.8 ou 10.9 com FL Studio Mac OS X Beta - Yosemite não suportado! -
